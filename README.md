# Olga Galkina — Portfolio

Портфолио графического дизайнера Ольги Галкиной.

## Как разместить на GitHub Pages

1. Создай новый репозиторий на GitHub (например: `portfolio`)
2. Загрузи файл `index.html` в корень репозитория
3. Перейди в Settings → Pages → Source: выбери `main` ветку, папку `/ (root)`
4. Нажми Save — через 1-2 минуты сайт будет доступен по адресу:
   `https://ВАШ_ЛОГИН.github.io/portfolio`
